// HOW I BUILT THIS

This vocab review game was built as an extra credit project for ICP (Introduction to Computer Programming) 2025–26, by Santiago Diaz.

  1. Got the assignment. The task was to vibe-code a vocab review game using an AI assistant, publish it on GitHub Pages, and submit the link.
  2. Chose my AI tool. I used Claude (Anthropic) — specifically Claude Sonnet — as my AI coding partner.
  3. Uploaded the word bank. I exported the ICP Vocab spreadsheet as a zip file from Google Sheets and uploaded it directly to Claude in the chat.
  4. Described the game. I told Claude I needed a multiple-choice quiz game that you can actually win, with my name on screen, a "How I Built This" modal, instructions, and a working win condition (3 lives, 15 questions).
  5. Claude generated the code. Claude extracted the vocab terms and definitions from the HTML export of my spreadsheet, then designed and coded the entire game — styling, logic, animations, modal, win/lose screens — all in a single index.html file.
  6. Published on GitHub Pages. I created a new GitHub repository, uploaded the index.html file, enabled GitHub Pages in the repo settings, and got a live link.

Tech used: HTML · CSS (custom properties, keyframe animations) · Vanilla JavaScript · Google Fonts (Orbitron, Share Tech Mono)
